# 90-Second Demo Script

1. Open landing page (`/`) and state: “SuiShield scans Move packages and emits deterministic receipts.”
2. Click **Scan** and choose **Deterministic Demo Scan**.
3. Submit scan. Mention that scanner uses fixed ruleset and canonical output ordering.
4. On `/r/[id]`, highlight:
   - score + verdict
   - grouped findings with evidence lines
   - `receipt_hash` and `ruleset_hash`
   - download receipt JSON
5. Go to `/verify`.
6. Load the same receipt by ID and run verify.
7. Show “Verified” result and hash comparison output.
8. Mention optional Sui attestation contract and that object hashes can be compared on-chain.
9. Close with: “This is open source, deterministic, and reproducible for judges.”
